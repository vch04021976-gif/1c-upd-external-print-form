# Внешняя печатная форма УПД со штампом для 1С:УТ 11.5

Данная разработка реализует формирование Универсального Передаточного Документа (УПД) Статус 1 для конфигурации "Управление торговлей 11.5" (проверено на версии 11.5.22.85).

## 🚀 Основные фичи:
- **Интеллектуальный поиск данных:** Обработка автоматически находит связанную и проведенную Счет-фактуру, даже если печать запущена из документа "Реализация товаров и услуг".
- **Прямой вызов сервисных модулей:** Реализован обход блокировок БСП через обращение к `Обработки.ПечатьОбщихФорм`. Это гарантирует работу там, где типовой метод `Печать()` менеджера документа возвращает пустоту.
- **Динамический штамп:** 
  - Поиск позиции "М.П." на листе методом `НайтиТекст` (не зависит от обновлений макета 1С).
  - Прозрачный фон печати (свойство `Узор = БезУзора`).
  - Жесткая фиксация пропорций (защита от сплющивания картинки).
  - Подавление вывода пустых вторых страниц.

## 🛠 Технические требования:
- Платформа 1С:Предприятие 8.3.21+
- Конфигурация: Управление торговлей 11.5 (БСП 3.1+)

## 📦 Установка:
1. Загрузите файл `.epf` в справочник "Дополнительные отчеты и обработки".
2. Назначьте команду для документов "Реализация товаров и услуг" и "Счет-фактура выданный".
3. В макете `МояПечать` (двоичные данные) внутри обработки замените файл на вашу печать.
